# 타마스터디 과제 모음

```
1. 과제 완료하면 체크박스에 체크
2. 이름 옆에 : (과제명? 과제주제이름? 등 무슨 프로젝트인지)
3. WEEK 폴더에 자신의 이름으로 폴더를 만들고, 그 안에 과제를 넣는다.
4. 이 과정이 끝나면 반드시 pull request한다.
```
